# 安全政策

## 支持的版本

我们目前维护最新版本的 SeeWxapkg。对于旧版本，我们仅提供重要的安全更新。

## 报告漏洞

如果您发现安全漏洞，请不要公开发布 Issue。

### 如何报告

请发送邮件至：[security@example.com](mailto:security@example.com)

邮件内容应包含：
- 漏洞描述
- 影响范围
- 复现步骤
- 建议的修复方案（如果有）

### 响应时间

我们会在收到报告后 48 小时内回复，并尽快修复问题。

## 安全最佳实践

1. **不要提交敏感信息**：切勿在代码中提交密码、密钥等敏感信息
2. **使用环境变量**：所有敏感配置都应通过环境变量配置
3. **定期更新依赖**：保持依赖包更新以获取安全修复
